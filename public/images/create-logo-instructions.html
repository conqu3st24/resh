Создание PNG логотипа RESHIP

Примечание: Это SVG-версия логотипа RESHIP. Вам нужно создать PNG-версию логотипа для использования в качестве резервного варианта.

Как создать PNG логотип:

  1. Сделайте скриншот логотипа из браузера (открыв этот HTML файл)
  2. Обрежьте изображение, оставив только логотип
  3. Сохраните файл как public/images/reship-logo.png
  4. Убедитесь, что логотип имеет правильный размер (примерно 120x60 пикселей)
Альтернативный вариант: Вы также можете использовать графический редактор (Photoshop, GIMP, Figma и т.д.) для создания логотипа по вашему усмотрению.
