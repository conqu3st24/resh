# Директория для изображений стримеров

Для корректной работы секции "Выбор стримеров" необходимо добавить следующие изображения:

- bratishkin.jpg - фотография Владимира Братишкина
- evelone.jpg - фотография Evelone

Также нужны изображения оборудования, которое они используют:
- logitech-keyboard.jpg - клавиатура Logitech G PRO X
- logitech-mousepad.jpg - коврик Logitech G PRO X

Рекомендуемые размеры изображений:
- Для стримеров: 668x411 пикселей
- Для оборудования: 250x87 пикселей

Временно в компоненте используются заглушки, но для production рекомендуется заменить их на реальные изображения.
