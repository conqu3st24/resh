# Директория для изображений инструментов

Для корректной работы секции "Инструмент для подбора" необходимо добавить следующее изображение:

- tool-background.jpg - фоновое изображение клавиатуры или другого игрового оборудования

Рекомендуемое разрешение изображения: 1163x494 пикселей

Временно в компоненте используется внешнее изображение с URL, но для production рекомендуется заменить его на локальное.

Внешнее изображение: https://maxgaming.com/img/cms/Keyboard/Keychron/Keychron%20V1/keychron_v1_qmk_via_custom_mechanical_keyboard_blue_1.jpeg
